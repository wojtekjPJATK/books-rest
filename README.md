## Installation

You can create a virtual environment and install the required packages with the following commands:

    $ virtualenv venv
    $ . venv/bin/activate
    (venv) \$ pip install -r requirements.txt

or for Windows:

```
virtualenv venv
venv/Scripts/activate
(venv) \$ pip install -r requirements.txt
```

## Running the Code

    (venv) $ python main.py
